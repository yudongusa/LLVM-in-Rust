## Project Overview

LLVM-in-Rust is a Rust implementation of LLVM support. The project is in early development with no source files yet.

## Common Commands

```bash
# Build
cargo build

# Build (release)
cargo build --release

# Run tests
cargo test

# Run a single test
cargo test <test_name>

# Lint
cargo clippy

# Format
cargo fmt

# Check without building
cargo check
```
